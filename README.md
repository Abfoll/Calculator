# React Calculator


## 🧮 Project Description
A sleek, responsive calculator built with React.js featuring:
- All basic operations (+, -, *, /)
- Decimal support
- Clear/delete functionality
- Modern UI with CSS Grid

## 🛠️ Technologies
- React.js (with useReducer)
- CSS Grid & Flexbox
- Vite (or Create-React-App)
